# 📊 Análisis de Desvíos Presupuestarios por Sucursal

Este proyecto implementa un flujo de control de gestión automatizado utilizando **Python** y la librería **Pandas**. El objetivo principal es optimizar la auditoría de costos mediante la consolidación y cálculo automatizado de variaciones presupuestarias, eliminando el procesamiento manual y reduciendo el margen de error operativo.

## 🛠️ Tecnologías Utilizadas
* **Lenguaje:** Python 3
* **Librerías Core:** Pandas (Estructuración y cálculo numérico), Matplotlib (Visualización de datos)
* **Entorno de Desarrollo:** JupyterLab (Anaconda Distribution)
* **Destino Operativo:** Exportación automatizada a formatos físicos nativos (.xlsx)

## 📈 Funcionalidades Clave
1. **Consolidación de Datos:** Procesamiento matricial de variables financieras complejas por unidades de negocio regionales.
2. **Cálculo de Variaciones:** Automatización del desvío crítico (`Gasto Real` vs `Presupuesto Asignado`) mediante lógica vectorial.
3. **Reportes Visuales:** Generación de gráficos analíticos de barras con segmentación cromática dinámica (ahorro en verde / exceso en rojo) para soporte en la toma de decisiones directivas.
4. **Gobierno de Datos:** Exportación directa a planillas Excel tradicionales listas para auditoría interna corporativa.
